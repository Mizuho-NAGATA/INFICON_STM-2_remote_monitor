# Contributing to STM2-remote / STM2-remote への貢献

## 日本語 (Japanese)

このプロジェクトへの貢献に興味をお持ちいただきありがとうございます！

### 貢献の方法

1. **Issue を作成する**
   - バグを発見した場合
   - 新しい機能を提案する場合
   - 質問がある場合

2. **プルリクエストを送る**
   - フォークしてブランチを作成
   - 変更を加える
   - テストを実行（該当する場合）
   - プルリクエストを作成

### コーディング規約

- PEP 8 スタイルガイドに従ってください
- 明確なコミットメッセージを書いてください
- コードには適切なコメントを含めてください

### 質問・サポート

質問がある場合は、Issue を作成してください。

## English

Thank you for your interest in contributing to this project!

### How to Contribute

1. **Create an Issue**
   - If you find a bug
   - If you want to propose a new feature
   - If you have questions

2. **Submit a Pull Request**
   - Fork and create a branch
   - Make your changes
   - Run tests (if applicable)
   - Create a pull request

### Coding Conventions

- Follow PEP 8 style guide
- Write clear commit messages
- Include appropriate comments in code

### Questions & Support

If you have questions, please create an Issue.
