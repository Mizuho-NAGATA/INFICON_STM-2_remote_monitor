# 📊 INFICON STM-2 リアルタイム遠隔監視システム
![外部向け説明資料](https://github.com/user-attachments/assets/eb5fce0c-8dbf-4847-b3d2-5c2d21164ab5)
**注意：このプログラムはINFICON社の公式なものではありません。**  
**Note: This program is not official INFICON.**  

本システムで使用する物性値について、正確性を保証するものではありません。利用により生じたいかなる結果についても、作者は責任を負いません。  
電子ビーム蒸着装置に取り付けられたSTM-2を想定して作成。  

## ✨ 特長

- STM-2 の膜厚・成膜レート・周波数をリアルタイム監視
- ネットワーク経由での多地点同時監視
- 目標膜厚の 80% を超えた際に自動アラート
- Docker の完全統合により環境構築が一切不要

# 📊 簡単セットアップガイド
## 🚀 **1分で理解！簡単3ステップ**

```
1️⃣ ダブルクリック → start_complete_system.bat
2️⃣ 画面の指示に従って待つだけ
3️⃣ GUIでファイルを選んで開始！
```

## 📋 **事前準備（初回のみ）**

### ✅ **必要なソフトウェア**
以下がインストールされていない場合、自動でインストール案内が表示されます：

1. **Python**（無料）
   - 🔗 https://www.python.org/downloads/
   - ⚠️ インストール時「Add Python to PATH」にチェック！

2. **Docker Desktop**（無料）
   - 🔗 https://www.docker.com/products/docker-desktop/
   - 💡 大きなファイルなのでダウンロードに時間がかかります

### ⚡ **ネットワーク設定**
- **固定IP推奨**：遠隔監視する場合は固定IPアドレスを設定してください
- **ファイアウォール**：自動で設定されます（管理者権限が必要）

---

## 🎯 **使い方（超簡単）**

### **【STEP 1】システム起動**
1. `start_complete_system.bat` をダブルクリック
2. 管理者権限の確認が出たら「はい」をクリック
3. **自動で全部起動します！** 

```
起動中の表示例：
[STEP 1/7] システム要件チェック中...
         ✓ Python インストール済み
         ✓ Docker Desktop インストール済み
[STEP 2/7] ファイアウォール設定中...
         ✓ ファイアウォール規則設定済み
...（続く）
```

### **【STEP 2】測定設定**
GUIが自動で開いたら：

1. **目標厚さ [nm]**：例）`100` を入力
2. **Material**：プルダウンから材料を選択（例：`Al`）
   - 選択すると密度とZ-ratioが自動入力されます
3. **ログファイル**：
   - 「参照」ボタンでSTM-2の`.log`ファイルを選択
   - または、ファイルをドラッグ&ドロップ

### **【STEP 3】監視開始**
1. **「Start Logging」**をクリック
2. ブラウザでGrafanaが自動で開きます
3. **リアルタイム監視開始！** 📊

### **【STEP 4】監視中の表示**
Grafanaダッシュボードで以下が表示されます：
- 📏 **現在の膜厚**
- 🎯 **目標膜厚**
- ⚡ **成膜レート**
- 🔔 **80%到達アラート**
- 📊 **周波数**

---

## 🛑 **停止方法**

### **通常の停止**
`stop_complete_system.bat` をダブルクリック

### **緊急停止**
1. GUIの「Stop Logging」をクリック
2. ブラウザを閉じる
3. 必要に応じて上記の停止スクリプトを実行

---

## 🆘 **困ったときのQ&A**

### **Q: 起動に失敗する**
**A: 以下を確認してください**
- インターネット接続は OK ですか？
- Python と Docker Desktop はインストール済みですか？
- 管理者権限で実行していますか？

### **Q: "No data" が表示される**
**A: データが来ていない可能性があります**
1. STM-2のログファイルが選択されているか確認
2. ログファイルが更新されているか確認
3. 「Stop Logging」→「Start Logging」で再開

### **Q: Grafanaが開かない**
**A: 手動でアクセスしてください**
- ブラウザで `http://localhost:3000` を開く
- ID: `admin` / パスワード: `admin`

### **Q: エラーメッセージが出る**
**A: スクリーンショットを撮って管理者に連絡**
- どの画面でエラーが出たか
- エラーメッセージの全文
- 何をしていたときに発生したか

## 🔧 **遠隔監視の設定**

### **他のPCから監視する場合**
1. STM-2接続PCのIPアドレスを確認
2. 監視したいPCのブラウザで以下にアクセス：
   ```
   http://[STM-2のIPアドレス]:3000
   ```
   例：`http://192.168.1.100:3000`

### **スマホから監視する場合**
同じネットワーク内であれば、スマホのブラウザからも監視可能です！

## 📱 **スマート機能**

### **自動アラート**
- 目標厚さの80%に到達すると自動で警告表示
- Grafanaのダッシュボードが赤色に変化

### **自動データ保存**
- すべてのデータは自動でデータベースに保存
- システム再起動後もデータは保持されます

### **マルチユーザー対応**
- 複数の人が同時に監視可能
- Run IDで実験を区別

## 🎓 **覚えておくと便利**

### **ファイル場所**
- ログファイル：STM-2ソフトウェアが作成する`.log`ファイル
- 設定保存：次回起動時に前回の設定が復元されます

### **データの見方**
- **厚さ**：nm（ナノメートル）単位で表示
- **レート**：nm/s（ナノメートル毎秒）で表示
- **80%アラート**：目標値の80%で自動警告

### **便利な機能**
- ドラッグ&ドロップでファイル選択可能
- 材料選択で密度・Z-ratioが自動入力
- Run IDはファイル名から自動生成
